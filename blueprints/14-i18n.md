# Blueprint 14 - Internationalisation (i18n)

## Vue d'ensemble
ArtisanCMS distingue deux types d'internationalisation :
1. **Interface admin** : traduction des libellés, boutons, messages (Laravel + React)
2. **Contenu multilingue** : traduction des pages, posts, menus (V2, prévu mais pas implémenté en V1)

En V1, seule la traduction de l'interface est implémentée. Le contenu est monolingue.

## 4. Partage avec React via Inertia

```php
// app/Http/Middleware/HandleInertiaRequests.php
public function share(Request $request): array
{
    return [
        ...parent::share($request),
        'locale' => app()->getLocale(),
        'translations' => fn () => $this->getTranslations(),
    ];
}

protected function getTranslations(): array
{
    $locale = app()->getLocale();
    $path = lang_path("{$locale}/cms.php");

    if (!file_exists($path)) {
        return [];
    }

    return require $path;
}
```

### Hook React useTranslation

```tsx
// resources/js/hooks/use-translation.ts
import { usePage } from '@inertiajs/react';

type TranslationKey = string;

export function useTranslation() {
    const { translations, locale } = usePage().props as {
        translations: Record<string, any>;
        locale: string;
    };

    /**
     * Traduire une clé (notation pointée)
     * Exemple : t('pages.title') → "Pages"
     * Avec remplacement : t('pages.messages.created', { name: 'Ma page' })
     */
    function t(key: TranslationKey, replacements?: Record<string, string>): string {
        const parts = key.split('.');
        let value: any = translations;

        for (const part of parts) {
            if (value && typeof value === 'object' && part in value) {
                value = value[part];
            } else {
                // Clé non trouvée, retourner la clé brute
                return key;
            }
        }

        if (typeof value !== 'string') return key;

        // Remplacements (ex: :name → "Ma page")
        if (replacements) {
            for (const [rKey, rValue] of Object.entries(replacements)) {
                value = value.replace(`:${rKey}`, rValue);
            }
        }

        return value;
    }

    return { t, locale };
}
```

## 5. Gestion de la locale

### Choix de la langue à l'installation
La langue est choisie à l'étape 2 du wizard d'installation et sauvegardée dans les settings.

```php
// app/Http/Middleware/SetLocale.php
<?php

declare(strict_types=1);

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class SetLocale
{
    public function handle(Request $request, Closure $next)
    {
        // 1. Préférence utilisateur (si connecté)
        $locale = $request->user()?->preferences['locale'] ?? null;

        // 2. Setting globale du CMS
        if (!$locale) {
            $locale = app(\App\Services\SettingService::class)->get('general.language', 'fr');
        }

        // 3. Vérifier que la locale existe
        $supported = ['fr', 'en', 'es', 'de', 'pt', 'it'];
        if (!in_array($locale, $supported)) {
            $locale = 'fr';
        }

        app()->setLocale($locale);

        return $next($request);
    }
}
```

### Langues supportées (V1)
| Code | Langue | Statut |
|------|--------|--------|
| `fr` | Français | Complet (langue par défaut) |
| `en` | English | Complet |

D'autres langues seront ajoutées progressivement ou via des contributions communautaires.

## 7. Contenu multilingue (V2 — futur)

> **Non implémenté en V1.** Documentation préparatoire.

### Approche envisagée : table de traductions

```sql
-- Table translations (V2)
CREATE TABLE translations (
    id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
    translatable_type VARCHAR(255),   -- App\Models\Page, App\Models\Post
    translatable_id BIGINT UNSIGNED,
    locale VARCHAR(5),                -- fr, en, es
    field VARCHAR(100),               -- title, slug, content, excerpt
    value JSON,                       -- Valeur traduite
    UNIQUE(translatable_type, translatable_id, locale, field)
);
```

### Impact sur l'UI
- Onglets par langue dans les formulaires d'édition
- Indicateur de traduction manquante
- Copier le contenu d'une langue vers une autre
- URL avec préfixe (`/fr/about`, `/en/about`) ou sous-domaine (`fr.site.com`)
